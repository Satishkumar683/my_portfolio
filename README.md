# 🌐 Personal Portfolio Website

A modern responsive developer portfolio website built using **HTML, CSS, and Vanilla JavaScript** showcasing projects, skills, competitive programming profiles, and achievements.

---

# 🚀 Features

* Modern futuristic UI design
* Fully responsive layout
* Smooth scrolling animations
* Glassmorphism design
* Interactive project cards
* Competitive programming section
* Skills & achievements showcase
* Contact section with social links
* Mobile-friendly navigation
* Animated hover effects

---

# 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Responsive Design
* Glassmorphism UI
